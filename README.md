# Setup

Two terminal tabs needed and one of these : LAMP XAMPP WAMP MAMP

## Back-end
From project root :
```bash
cd back-end
```

Install dependencies.

```bash
npm install
composer install
```

Lanch API server

```bash
php artisan serve
```

## Front-end

From project root :
```bash
cd front-end
```

Install dependencies.

```bash
npm install
```

Lanch quasar

```bash
quasar serve dist/pwa
```
